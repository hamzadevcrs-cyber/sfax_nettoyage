ENDPOINTS API
================================================================================

GET  /api/interventions              - Jälä toutes les interventions
GET  /api/interventions?date=...     - Filtre par date
GET  /api/interventions?municipalite=... - Filtre par commune
GET  /api/interventions/dates        - Dates disponibles
GET  /api/stats                      - Statistiques globales
GET  /api/stats/by-municipality      - Stats par commune
GET  /api/stats/by-date              - Stats par date
POST /api/interventions              - Créer une intervention
GET  /api/health                     - Vérifier serveur

DÉMARRAGE RAPIDE
================================================================================

1. Installation:
   $ npm install

2. Vérification des variables:
   $ cat .env
   # Doit contenir: VITE_SUPABASE_URL et VITE_SUPABASE_ANON_KEY

3. Lancement:
   $ node server.js
   
4. Accès:
   http://localhost:3000

5. Test:
   $ curl http://localhost:3000/api/health

   → Vérifier: .env avec bonnes credentials
   → Vérifier: connexion Internet
   → Vérifier: quota DB pas dépassé

   → Vérifier: F12 → Console → Erreurs?
