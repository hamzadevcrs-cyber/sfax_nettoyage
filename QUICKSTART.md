# البدء السريع (Quick Start)

## في دقيقة واحدة

### الخطوة 1: تثبيت

```bash
npm install
```

### الخطوة 2: التشغيل

```bash
node server.js
```

### الخطوة 3: الوصول

افتح المتصفح: `http://localhost:3000`

---

## ماذا يحدث الآن؟

### الواجهة الأمامية
```
✓ لوحة المتابعة (Dashboard)
  - الإجماليات (KPIs)
  - الرسوم البيانية
  
✓ البلديات (Municipalities)
  - إحصائيات كل بلدية
  - الترتيب والتصفية
  
✓ الكميات (Quantities)
  - جدول تفصيلي
  - البحث حسب التاريخ
```

### الخادم الخلفي
```
✓ API مع 7 endpoints
  - GET /api/interventions (جلب البيانات)
  - POST /api/interventions (إضافة تدخل)
  - GET /api/stats (الإحصائيات)
  - + المزيد
  
✓ قاعدة بيانات Supabase
  - تخزين آمن
  - Row Level Security
  - النسخ الاحتياطية
```

## الميزات

### 1. إضافة تدخل جديد
```
1. اضغط الزر "➕ إضافة تدخل"
2. ملأ النموذج:
   - اختر التاريخ والبلدية
   - أدخل الكمية (طن)
   - حدد نوع التدخل
   - أضف الأماكن والموارد
3. اضغط "حفظ"
4. سيتم تحديث اللوحة تلقائياً
```

### 2. عرض البيانات
```
- لوحة المتابعة: رسوم بيانية ومؤشرات أساسية
- البلديات: بطاقات لكل بلدية
- الكميات: جدول تفصيلي كامل
```

### 3. التصفية والترتيب
```
- حسب البلدية
- حسب التاريخ
- ترتيب تصاعدي/تنازلي
- إعادة تعيين الفلاتر
```

### 4. التصدير
```
- اضغط "⬇ تصدير JSON"
- سيتم تحميل ملف يحتوي على:
  - الإحصائيات
  - البيانات الكاملة
  - التاريخ والوقت
```

## المشاكل الشائعة

### ❌ "لا يمكن الاتصال بـ localhost:3000"
```
✓ تحقق أن الخادم يعمل:
  - اضغط Ctrl+C وأعد التشغيل: node server.js
  - تأكد من عدم استخدام المنفذ 3000 من برنامج آخر
```

### ❌ "خطأ: لا توجد بيانات Supabase"
```
✓ تحقق من ملف .env:
  - VITE_SUPABASE_URL=... (يجب أن يكون موجوداً)
  - VITE_SUPABASE_ANON_KEY=... (يجب أن يكون موجوداً)
  - أعد التشغيل بعد التعديل
```

### ❌ "خطأ عند حفظ التدخل"
```
✓ تحقق من:
  - هل أدخلت الكمية (> 0)؟
  - هل اخترت البلدية؟
  - هل ملأت نوع التدخل؟
  - هل ملأت الأماكن؟
```

### ❌ "الرسوم البيانية لا تظهر"
```
✓ افتح F12 وتحقق من:
  - console للأخطاء
  - Network للطلبات الفاشلة
  - تأكد من تحميل Chart.js من CDN
```

## الدعم السريع

### هل تحتاج مساعدة؟

1. **تحقق من السجلات:**
   ```bash
   # اضغط Ctrl+C ثم
   node server.js
   # اقرأ الأخطاء بعناية
   ```

2. **افتح وحدة تحكم المتصفح:**
   ```
   F12 → Console tab
   اقرأ رسائل الخطأ
   ```

3. **تحقق من الاتصال:**
   ```bash
   curl http://localhost:3000/api/health
   # يجب أن تأتي استجابة "ok"
   ```

4. **أعد التشغيل:**
   ```bash
   # أغلق الخادم: Ctrl+C
   # ثم أعده:
   node server.js
   ```
